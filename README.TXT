woWop

Qt based MongoDB browser
Waitman Gobble <waitman@waitman.net>

NO WARRANTIES

This example iterates through dbs/collections on local running MongoDB and 
lists in tree view on left pane of Splitter Widget, when the user clicks
on a collection it does a db.find() and shows results converted to HTML 
in a QtWebKit widget on the right pane of the Splitter Widget.
